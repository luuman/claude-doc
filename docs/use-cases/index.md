## 💡 通用建议

无论选择哪个案例，遵循这些原则能获得最佳效果：

1. **明确目标** — 在启动 Claude Code 前，清楚描述你想要什么
2. **提供上下文** — 给出足够的背景信息（架构、约束条件、标准）
3. **迭代优化** — 第一次可能不完美，基于反馈迭代调整
4. **自动化重复** — 将成功的工作流封装为 Skills 或 Hooks
5. **团队共享** — 将有效的方案分享给团队，标准化工作流
